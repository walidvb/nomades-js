Outils

Une instruction se termine toujours par un ;
Variables: var foo;

Type: 3 types de variables: sans type, Array , Object

Assignation: 

foo = 3 + 2;
foo = 'a string';
myBook = {
	ISBN: 123456,
	author: 'Albert Camus',
	title: "L'étranger", 
};

Fonctions: function bar(){};

Une fonction prend des arguments en entrée, et retourne une valeur.

Déclaration: 

function square(a) {
	//corps de la fonction
	var result = a*a;
	return result;
}

Appel: square(3);

Portée: une variable créée à l'intérieur d'une fonction n'existe qu'à l'intérieur de cette fonction.

Effet de bord: Une fonction ne peut pas affecter le code qui se trouve en dehors de son corps.

Structure de controles: if(...){} , for(...){} , switch(...){} , ...

if:

if(condition1){ 
	instructions; 
}
else if(condition2){
	instructions; 

}
else{
	instructions; 
}

Les conditions sont des variables booléennes, qui valent soit true , soit false

Opérateurs sur les booléens:

  • a == b : a est égal à b
  • a != b : a est différent de b
  • a > b / a >= b / a < b / a <= b : a est plus strictement plus grand/ plus grand ou égal / strictement plus petit / plus petit ou égal à b
  • a || b : a ou b
  • a && b : a et b

for:

for permet de répéter une opération plusieurs fois. il prend comme paramètres une initialisation( var i = 0; ), une condition d'arrêt ( i < 10 ), et une instruction d'incrémentation( i++ ) 

for(var i = 0; i < 10; i++)
{
	instructions;
}
Existent encore do{...} while , switch(foo){ ... } , ...
L'instruction break; permet de sortir d'une structure de controle
Fourni par javascript
  • alert("Alert!"); ouvre un dialogue d'alerte(à éviter!)
  • var name = prompt("What's your name?"); ouvre un dialogue d'alerte, et assigne l'entrée de l'utilisateur à une variable
  • console.log("foo is now: ", foo); écrit dans la console(utile pour débugger)
  • L'objet Math fournit les opérations standards mathématiques( Math.min(3, 4) = 3 , Math.sqrt(9) = 3 , ...)
  • typeof foo permet de déterminer le type d'objet de foo
