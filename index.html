une fonction qui s'appelle elle-même est une fonction récursive!
Exemple: Calculer n! Rappel: ( n! = n*(n-1)*(n-2)*...*1 ) 
function factorial(num)
{
    // If the number is less than 0, reject it.
    if (num < 0) {
        return -1;
    }
    // If the number is 0, its factorial is 1.
    else if (num == 0) {
        return 1;
    }
    // Otherwise, call this recursive procedure again.
    else {
        return (num * factorial(num - 1));
    }
}

Les conditions sont des variables booléennes, qui valent soit true , soit false

Comment créer des booléens?:

  • a = true;
  • a == b : a est égal à b
  • a != b : a est différent de b
  • a > b / a >= b / a < b / a <= b : a est plus strictement plus grand/ plus grand ou égal / strictement plus petit / plus petit ou égal à b
  • a || b : a ou b
  • a && b : a et b

Objects

En javascript, presque tout est un object.

un Object permet de stocker plusieurs donnée dans une variable: 

var walid = {
	name : 'Walid',
	address: '3, rue du Faubourg'
}

un Object peut posséder, en plus de données, des fonctions! 

var person = {
	name : 'Walid',
	address : '3, rue du Faubourg',
	getInfo : function(){
		return this.name + ' ' + this.address;
	}
}

On peut accéder aux propriétés des Object s en appelant Object.property ou Object['property']

les Object s fournis par javascript possèdent déjà leurs propres méthodes!

  • String : .toLowerCase() , .length
  • Array : myArray[3] , .push(elem) (pour ajouter elem dans l'array), .indexOf(elem) (permet de connaître l'emplacement de elem dans l'array. Si elem n'est pas trouvé, retourne -1)
  • console possède la méthode log !
  • document : getElementById(id) , getElementsByClass(id)

On peut créer un constructeur d'objet, de manière à créer un 'classe': 

function Person(name, address) {
	this.name = name;
	this.address = address;
	this.getInfo = function() {
		return this.name + ' ' + this.address;
	}
	return this;
}

var walid = new Person('Walid', '3, rue du Faubourg');
var michel = new Person('Michel', '18, col du Simplon')

console.log(walid.address) //loggera '3, rue du Faubourg'

Syntaxe II

Manipulation du HTML

Pour récupérer des éléments de la page html, les différentes fonctions sont disponibles:

  • document.getElementById(id); : récupérer l'élément "#id"
  • document.getElementsByClassName(class); : récupérer la liste des éléments avec la classe "class"
  • document.getElementsByTagName(tag); : récupérer la liste des éléments de type "tag"

Les fonctions getElementsByClassName() et getElementsByTagName() retourne une liste!

Pour accéder au premier élément trouvé, il est donc nécessaire d'utiliser la syntaxe des Array : document.getElementsByClassName('my-class')[0]; !

Pour modifier le contenu de ces éléments: element.innerHTML = "nouveau contenu!;"

Pour lire le contenu d'un élément input : var value = document.getElementById('#input').value;

Pour modifier le style d'un élément: element.style.color = 'blue'; element.style.['backgroud-color'] = 'blue';
